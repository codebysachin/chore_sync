# KinKeeper - Family Chore Management App

KinKeeper is a React Native application designed to streamline household chore management and foster family collaboration through an intuitive, engaging interface. The app transforms everyday household tasks into a coordinated family effort, promoting responsibility and teamwork among family members.

## Features

### Core Functionality
- **Task Management**: Create, assign, and track household chores with ease
- **Family Profiles**: Individual profiles for each family member with customizable avatars
- **Schedule Organization**: Weekly and monthly task planning with automated reminders
- **Progress Tracking**: Visual representations of task completion and contribution metrics
- **Reward System**: Built-in reward mechanism to motivate family members

## Development Setup

### Prerequisites
Make sure you have completed the [React Native - Environment Setup](https://reactnative.dev/docs/environment-setup) instructions before proceeding.

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/kinkeeper.git
```

2. Install dependencies
```bash
npm install
# OR
yarn install
```

### Running the App

1. Start Metro Server
```bash
npm start
# OR
yarn start
```

2. Run the Application

For iOS:
```bash
npm run ios
# OR
yarn ios
```

For Android:
```bash
npm run android
# OR
yarn android
```

## Technical Stack
- React Native for cross-platform mobile development
- Redux for state management
- AsyncStorage for local data persistence
- Push Notifications for task reminders
- User Authentication for family account security
